## What it does

Ingests PDF quarterly reports plus markdown notes, extracts the structure, and produces a polished interactive HTML dashboard you can host or email.

## Why it exists

Every quarter I was rebuilding the same review deck by hand. Now it's a one-command refresh.

## Stack

Python · PDF extraction · Jinja2-style HTML templating · static output (host anywhere for free).
